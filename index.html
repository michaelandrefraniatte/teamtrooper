TeamTrooper

Helping Team Projects

Program for Managing Projects with Teammates in Groups

This program is a diagram for managing projects and files associated, with teammates in groups, in a local network, and it's also a proof of concept of a new structure of database, allowing programing smarter. Let's see...

Considerations About the Program

It needs at least 2 administrators, even with same account authenticated. The administrators are setting root folder, projects and teammates in groups for initialize it. You must set the root folder with credentials (computer name, and session password) from the file named teamtrooper.txt. For be able to open the program it need to create the local authentication file by just opening the program for it under same folder. This for each PC you want to use TeamTrooper and each time TeamTrooper change. My source codes and program are here.
